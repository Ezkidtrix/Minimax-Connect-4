# Minimax Connect 4
A game of Connect 4, with either an opponent of someone else like a friend, or a bot using the minimax algorithm.

## Instructions
Using the dropdown menu, you can click either of the 3 difficulties, or just play against someone else with you, just take turns using the device as it's not online multiplayer. You can click any of the 7 columns to drop a piece in that column. When someone connects 4 in a row, they will win. If it's a draw, it will say that it's a draw and you can restart.
